# Print Hello world to the Screen

>## 1. Create a new directory called Hello world and get inside that directory

```cmd
    mkdir Hello-world
    cd Hello-world

```

> ## 2. Create a main.rs file and put the following code

```rust
fn main(){
    println!("Hello world")
}
```

> ## 3. compile the main.rs file

```cmd
    rustc main.rs
    .\main.exe
```
